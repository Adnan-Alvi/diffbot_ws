# Autonomous Differential Drive Robot | ROS 2 Humble

Complete ROS 2 Humble setup for an autonomous differential drive robot (workspace: `diffbot_ws`).  
Works out of the box with Ignition Gazebo.

## Features
- Differential drive robot model with sensors (URDF/Xacro)
- Ignition Gazebo simulation
- Odometry and proper motion control
- Mapping with SLAM Toolbox
- Localization with AMCL
- Full autonomous navigation using Nav2
- Everything ROS 2 Humble compatible

Just clone, build, and run. No extra junk.
